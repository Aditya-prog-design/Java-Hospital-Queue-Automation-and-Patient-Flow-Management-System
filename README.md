# Java Hospital Queue Automation and Patient Flow Management System

## 📌 Overview

This project is a Java-based application designed to automate hospital operations like patient registration, queue management, and record handling. It reduces manual work and improves efficiency in hospitals.

---

## 🎯 Features

* Patient Registration System
* Automated Queue Management (FIFO & Priority Queue)
* Doctor Assignment
* Appointment Scheduling
* Patient Record Management
* Simple GUI using Java Swing

---

## 🛠️ Technologies Used

* Java (Core + OOP)
* Java Swing (GUI)
* MySQL (Database)
* JDBC (Database Connectivity)

---

## ⚙️ How It Works

1. Patient registers into the system
2. Data is stored in the database
3. Patient is added to queue
4. Doctor calls next patient
5. Treatment is completed
6. Records are updated

---

## 💻 Installation & Setup

1. Clone the repository

   ```bash
   git clone https://github.com/Aditya-prog-design/Java-Hospital-Queue-Automation-and-Patient-Flow-Management-System.git
   ```

2. Open project in Eclipse / IntelliJ

3. Compile and run:

   ```bash
   javac HospitalQueueSystem.java
   java HospitalQueueSystem
   ```
